Система детектирования забытых рюкзаков

Веб-приложение для автоматического обнаружения забытых рюкзаков в аэропорту с использованием нейронных сетей.

Функционал

- Загрузка изображений - детектирование рюкзаков на фото
- Обработка видео - анализ видеофайлов
- Веб-камера - детектирование в реальном времени
- История запросов - сохранение результатов в БД
- Генерация отчетов - PDF и Excel форматы
- Современный интерфейс - нежно-розовый дизайн

Технологии

- Backend: Python 3.9+, Flask
- Модель: YOLOv8 (Ultralytics)
- Компьютерное зрение: OpenCV
- База данных: SQLite
- Фронтенд: HTML5, CSS3, JavaScript
- Отчеты: ReportLab, openpyxl
